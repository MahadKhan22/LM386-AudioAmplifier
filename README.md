# LM386-AudioAmplifier
Latex Circuit diagram and LTspice schematic for my end-semester Electrical-Network-Analysis project

The LM386 is not a built-in component in LTspice, so you'll need the .MOD & .ASY files in the same directory as the .asc file for the simulation to work.

The default gain on this chip is 20 (26dB), but can be varied between 20 and 200 by configuring capacitors and resistors on pins 1 & 8.
